# Ostrze–Cierpliwość – Agresja zmian (architektura kwantowa)

W architekturze kwantowej parametr **Ostrze–Cierpliwość** ma odniesienie do liczby iteracji obwodu kwantowego i liczby pomiarów.  
`step_size` może oznaczać głębokość korekcji w obwodzie (np. liczbę warstw variational ansatz), a `memory` – liczbę powtórzeń pomiaru, z których uśredniamy wynik.  
`hysteresis` definiuje próg minimalnej różnicy energii lub funkcji celu, poniżej którego nie wprowadzamy zmian w obwodzie.
