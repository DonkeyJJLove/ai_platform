# Wioska–Miasto – Skala społeczna (architektura kwantowa)

W architekturze kwantowej rejestr Wioska–Miasto wpływa na to, ile rejestrów kubitów jest jednocześnie wykorzystywanych.  
`scope=local` może oznaczać pojedynczy rejestr z ograniczoną liczbą kubitów; `scope=global` – dystrybucję obliczeń na wiele rejestrów i synchronizację wyników pomiarów.  
Parametr `aggregation` wskazuje, czy wyniki pomiarów są uśredniane, czy wybierany jest wynik o największej amplitudzie.  
`fanout_limit` w tej architekturze odnosi się do liczby kanałów klasycznych, do których przekazywana jest superpozycja wyników.
