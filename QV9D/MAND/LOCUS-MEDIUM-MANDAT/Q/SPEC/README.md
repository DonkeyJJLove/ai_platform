# Locus–Medium–Mandat – Zakotwiczenie stanu (architektura kwantowa)

W architekturze kwantowej rejestr **Locus–Medium–Mandat** obejmuje zarówno klasyczne repozytoria, jak i rejestry kubitów w pamięci kwantowej.  
`location` może określać adres na klasycznym klastrze oraz identyfikator rejestru kubitów.  
`medium` obejmuje formaty takie jak `qasm` (program obwodu), `yaml` czy `bin`.  
`authoritative` wskazuje, czy zmiana parametru obwodu wymaga akceptacji człowieka (np. `human`, `hybrid`), czy może być dokonana autonomicznie (`ai`).
