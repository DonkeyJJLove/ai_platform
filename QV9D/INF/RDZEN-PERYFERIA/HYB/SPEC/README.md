# Rdzeń–Peryferia – Priorytety obszarów (architektura hybrydowa)

W hybrydzie parametr Rdzeń–Peryferia decyduje nie tylko o podziale matrycy ASCII, ale również o tym, które kafelki mozaiki oraz węzły AST otrzymują większy budżet obliczeniowy.  
Pola `core_size`, `periphery_size` i `redundancy` interpretuje się analogicznie jak w wersji klasycznej.  
Parametr `layout` może być rozszerzony o mapowanie kafelków na domeny semantyczne, co pozwala hybrydowemu agentowi kierować uwagę tam, gdzie przyniesie największy zysk semantyczny.
